Selenium Interview Questions and Answers

This page contains the most commonly asked Selenium interview questions for freshers and experienced automation testers.

1. What is Selenium?

Selenium is an open-source automation testing tool used to test web applications across different browsers.

2. What are the components of Selenium?

  • Selenium WebDriver
  • Selenium IDE
  • Selenium Grid

3. What is WebDriver?

WebDriver is a tool that automates browser actions by directly communicating with browser drivers.

4. Difference between implicit and explicit wait?

Implicit wait applies globally, while explicit wait waits for a specific condition.

5. What is Page Object Model?

POM is a design pattern where each web page is represented as a class.

6. How do you handle dropdowns?

Using the Select class in Selenium.

7. How do you handle alerts?

Using driver.switchTo().alert()

8. What is TestNG?

TestNG is a testing framework used with Selenium for test execution and reporting.

9. How do you run tests in parallel?

Using TestNG XML configuration.

10. How do you handle dynamic elements?

Using dynamic XPath and waits.

Tip: These questions are frequently asked in QA automation interviews.
